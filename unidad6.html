CAPITULO 6: EVOLUCIÓN DE LOS SISTEMAS CELULARES

EVOLUCIÓN DE LOS SISTEMAS CELULARES


SISTEMA IMT-2000

Características

  1. Unidades terminales robustas, que requieren poco mantenimiento y soportan condiciones ambientales muy diferentes
  2. Flexibilidad y modularidad.
  3. Cobertura de grandes territorios
  4. Altos niveles de fiabilidad y disponibilidad del equipo

UMTS (Universal Mobile Telecommunications System)

Características

  1. Uso de tecnología WCDMA (Wide Code Division Multiple Access)
  2. Velocidad de acceso a Internet elevada
  3. Transmisión de audio y video en tiempo real
  4. Transmisión de voz con calidad equiparable a la de las redes fijas

WIN (Wireless Intelligent Network)

Características

  1. Servicios como identificador de llamadas y mensajería de voz incluidos en el paquete cuando compran y activan un teléfono celular o de servicio de comunicaciones personales (PCS)
  2. Emplea red de señalización existente SS7

SERVICIOS WIN

Servicios de Roaming

  1. Permite identificar a usuarios cuando están fuera de su área de servicio y poder ofrecerle el servicio

Selección de operador

  1. Permite que un usuario con el mismo equipo pueda tener el servicio contratado con diferentes operadores a la vez

Marcación por reconocimiento de voz

  1. Permite al usuario manejar su equipo móvil con comandos de voz. Esta es una función residente en los equipos que se complementa con la red WIN

Tarjetas prepagadas

  1. Permite hacer llamadas pagando el servicio con anterioridad

Métodos de tarificación

  1. Permite aplicar diferentes tarifas a los usuarios según la originación de la llamada, el destino, el contrato y el servicio

Servicios de transmisión de datos

  1. Permite ver al teléfono como un pager y conjuntar funciones, como llamar al número que envió el mensaje, etc.

MOBILE IP

Mobile IP es un estándar de protocolo comunicación diseñado para permitir a usuarios de dispositivos móviles moverse entre redes manteniendo la misma dirección IP.

Tipos de Movilidad

  1. Macro movilidad: el desplazamiento de un móvil se produce entre dos dominios de redes diferentes.
  2. Micro movilidad: el desplazamiento de un móvil se produce entre dos subredes de un mismo dominio administrativo.

WAP (Wireless Application Protocol)

Características

  1. Estándar de facto o un protocolo diseñado para micro-navegadores, y permite que los dispositivos móviles interactúen, intercambien y transmitan información a través de Internet
  2. Consta de 3 niveles conocidos como Cliente, Gateway y Servidor de origen
  3. La solicitud que envía a través del móvil a la puerta de enlace WAP se denomina solicitud de codificación.
